## Development

*Only once:*

- Open Folder with Visual Studio
- Open Pakcage Manager and run:
  ```sh
  Add-Migration v1.0.0
  ```
- run
  ```sh
  Update-database
  ```

**Note: _Check that your database is running, otherwise you are going to face issues_**

*Steps:*
- Run Visual studio with IIS Express
